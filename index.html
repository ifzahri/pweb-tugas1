Forms

HTML memiliki tag sendiri untuk mengontrol sekaligus mengumpulkan input text, beberapa diantaranya adalah form, textarea, input, dan lainnya. Dibawah ini adalah contoh form pada HTML.

Input ini memiliki tipe email, dimana dikhususkan untuk menyimpan format email dan bisa dilakukan modifikasi khusus.
Input ini memiliki tipe password. Tipe ini dirancang agar ketika pengguna memasukkan inputan, maka karakter password yang bersifat rahasia akan diganti menggunakan titik-titik untuk menjaga privasi.
Ketika dihadapkan dengan beberapa pilihan pada inputan, developer dapat menggunakan tag select yang menggabungkan pilihan menjadi satu dropdown. Masing-masing baris memiliki nilai tersendiri yang tentunya meski menjadi satu inputan, komputer akan mengenali nilai-nilai yang berbeda pada masing-masing pilihan.
Radio memiliki kemiripan dengan checkbox. Radio dipakai ketika pengguna diharuskan untuk memilih satu dari beberapa pilihan
Textarea pada sebuah form umum dipakai untuk menampung sebuah data yang lebih panjang dan memungkinkan pengguna untuk menulis secara multi-line dan menekan tombol ENTER.
Button submit diatas ditujukan untuk mengirim data yang telah ditampung pada form, dan dikirimkan sesuai dengan path dan HTTP method (GET, POST, PUT, PATCH, dan lainnya) yang ditentukan pada awal tag form.
